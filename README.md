## 路线图
1. 使用 GitBook 生成 GitHub Pages，作为实际发布页面。
2. 通过 Gitee 等平台建立国内镜像。
3. 通过知乎专栏等形式在社区进行发布。
4. 添加 `Q_D`/`D_Q` 私有指针、CorePrivate 模块等未在官方文档中正式发布，但已作为事实标准长期使用的特性介绍。
