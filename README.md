## 介绍

本项目本意是来根据官方文档翻译对应的中文文档

## 格式

每一个类新建一个文件夹。

从A-Z总共12个父文件夹

## 举例

比如我想翻译 QX11Info 类。

我要在X文件夹下，新建一个QX11Info的文件夹。

然后把 对应的 .md文档 放进去


# Import

各种提交代码之前一定要

```shell
git pull --rebase
```

# 一个简单的提交模板

Template.md 文件在当前根目录，可以根据这个模板往里添加对应的翻译，抑或参考其它的得翻译文档
